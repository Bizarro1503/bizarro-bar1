1 Crear Cuenta en GitHub

Si ya tienes cuenta, salta al paso 2.

🌐 Ir a: https://github.com/signup

📝 Completa:
• Email
• Contraseña
• Nombre de usuario

✉️ Verifica tu email
💡 Consejo: Elige un nombre de usuario profesional, será parte de tu URL: tunombre.github.io
2 Crear Nuevo Repositorio

Un repositorio es como una carpeta en la nube para tu proyecto.

➕ Clic en botón verde "New" o "+"

📝 Seleccionar: "New repository"

✍️ Nombre del repositorio: bizarro-bar

🌍 Marcar: "Public"

✅ NO marcar "Add a README file"

🚀 Clic: "Create repository"
⚠️ Importante: El nombre del repositorio será parte de tu URL. Usa solo letras minúsculas y guiones.
3 Subir Archivos

Ahora vamos a subir tu aplicación completa.

📤 Verás una página que dice:
"uploading an existing file"

👆 HAZ CLIC en ese enlace

📂 Se abrirá un área que dice:
"Drag files here to add them to your repository"
✅ Qué Arrastrar: Selecciona TODOS los archivos y carpetas del proyecto Bizarro Bar
Selecciona y arrastra estos elementos:
index.html
manifest.json
README.md
Carpeta css/ (completa)
Carpeta js/ (completa)
🖱️ ARRASTRA los 5 elementos juntos
a la ventana de GitHub

⏳ Espera que suban (barras de progreso)

💬 Abajo verás: "Commit changes"

📝 (Opcional) Escribe: "Primera versión del sistema"

✅ Clic: "Commit changes"
💡 Tip: GitHub mantiene la estructura de carpetas automáticamente. No necesitas hacer nada especial.
4 Verificar Archivos

Asegúrate de que todo se haya subido correctamente.

Debes ver esta estructura en GitHub:

📄 index.html ✅
📄 manifest.json ✅
📄 README.md ✅
📁 css ✅
📁 js ✅
✅ Perfecto: Si ves estos 5 elementos, todo está correcto.
⚠️ Si falta algo: Puedes volver a subir los archivos faltantes con "Add file" → "Upload files"
5 Activar GitHub Pages

Este paso hace que tu aplicación sea accesible desde internet.

⚙️ Clic en: "Settings"
(pestaña arriba, junto a Code)

📄 En el menú lateral izquierdo:
Busca y clic en: "Pages"

🌿 En "Source" selecciona: "main"

📁 En "Folder" deja: "/ (root)"

💾 Clic: "Save"
🎉 ¡Listo! GitHub tardará 1-2 minutos en publicar tu sitio.
6 Obtener tu URL

Tu aplicación ya está en línea. Ahora vamos a encontrar la URL.

🔄 Recarga la página de Settings → Pages

Verás un mensaje verde:
"Your site is published at:"

🔗 Tu URL será algo como:
https://tu-usuario.github.io/bizarro-bar/
# Ejemplo de URL:
https://juanperez.github.io/bizarro-bar/

# Formato:
https://[tu-usuario].github.io/[nombre-repo]/
